# DrawGD
This program is a tool for transferring bitmap images to Geometry Dash levels.

## Advantages
- Built-in picture editor
- Supported export to .GMD file

## Program interface
- Languages: English

## Hot Keys
- CTRL + V - Paste an image from the clipboard
